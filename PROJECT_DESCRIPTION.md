# 噪音实时反馈系统 (Noise Revenger)

## 项目背景

楼上住户产生的噪音严重影响日常生活，主要包括两类：
1. **低频冲击噪音**：砸楼板、重物落地、用力走路等
2. **高频摩擦噪音**：拉拽凳子、拖动家具等

多次反馈无果，因此开发一套自动化噪音实时反馈系统，在检测到噪音时自动播放提醒声音，实现即时反馈。

## 系统概述

本系统运行在 Windows 电脑上，通过麦克风实时采集环境音频，使用信号处理算法识别噪音事件，一旦检测到符合条件的噪音，立即通过蓝牙音箱播放预设的提醒音频。

## 硬件需求

| 设备 | 说明 |
|------|------|
| Windows 电脑 | 运行本系统 |

| 蓝牙音箱 | 播放提醒声音 |

## 核心功能

### 1. 实时音频采集
- 持续从麦克风采集环境音频流
- 支持配置采样率、缓冲区大小
- 低延迟处理

### 2. 噪音检测引擎

#### 2.1 低频冲击噪音检测
- 检测砸楼板、重物落地、用力走路等噪音
- 基于 FFT 频谱分析，重点监测 20Hz - 250Hz 频段
- 使用短时能量 + 频谱质心特征
- 可配置阈值和灵敏度

#### 2.2 高频摩擦噪音检测
- 检测拉拽凳子、拖动家具等尖锐摩擦声
- 重点监测 2kHz - 8kHz 频段
- 使用频谱突变检测 + 过零率特征
- 可配置阈值和灵敏度

#### 2.3 通用特征
- 自适应背景噪声学习（启动时学习当前环境底噪）
- 冷却时间机制（防止短时间内重复触发）
- 噪音强度分级（轻度/中度/重度）

### 3. 反馈播放模块
- 检测到噪音后自动播放提醒声音
- 支持多种提醒音效（提示音、语音警告等）
- 根据噪音强度选择不同音效
- 支持自定义音频文件
- 蓝牙音箱自动输出

### 4. 配置管理
- JSON/YAML 配置文件
- 可调节参数：
  - 检测阈值
  - 灵敏度
  - 冷却时间
  - 音频设备选择
  - 提醒音效选择

### 5. 日志与统计
- 记录每次噪音检测事件
- 包含时间戳、噪音类型、强度等级
- 生成每日/每周噪音报告
- 可选：保存噪音音频片段用于回溯

### 6. 用户界面（可选，V2）
- 系统托盘运行
- 实时显示噪音波形/频谱
- 手动控制开关
- 查看历史记录

## 关键算法说明

### 低频冲击噪音检测

```
1. 对音频帧进行 FFT 变换
2. 提取 20Hz-250Hz 频段能量
3. 计算短时能量 (STE)
4. 计算频谱质心 (Spectral Centroid)
5. 判断条件：
   - 低频能量 > 阈值 × 背景噪声
   - 频谱质心 < 500Hz（能量集中在低频）
   - 能量突增（当前帧能量 > 前N帧均值 × 倍数）
```

### 高频摩擦噪音检测

```
1. 对音频帧进行 FFT 变换
2. 提取 2kHz-8kHz 频段能量
3. 计算过零率 (Zero Crossing Rate)
4. 计算频谱平坦度 (Spectral Flatness)
5. 判断条件：
   - 高频能量 > 阈值 × 背景噪声
   - 过零率高（尖锐声音特征）
   - 频谱平坦度低（非白噪声，有明确频率成分）
```

## 配置示例

```yaml
# settings.yaml

audio:
  sample_rate: 44100
  chunk_size: 1024
  channels: 1
  input_device_index: null  # null = 默认设备

detection:
  # 低频冲击检测
  low_freq:
    enabled: true
    freq_range: [20, 250]
    energy_threshold: 3.0      # 相对于背景噪声的倍数
    spectral_centroid_max: 500 # Hz
    window_size: 0.1           # 秒

  # 高频摩擦检测
  high_freq:
    enabled: true
    freq_range: [2000, 8000]
    energy_threshold: 2.5
    zero_crossing_min: 0.3
    window_size: 0.05

  # 通用设置
  cooldown_seconds: 5          # 冷却时间
  background_learning_seconds: 10  # 背景噪声学习时间

feedback:
  enabled: true
  output_device_index: null    # null = 默认设备
  sounds:
    mild: "sounds/alert_mild.wav"
    medium: "sounds/alert_medium.wav"
    strong: "sounds/alert_strong.wav"
  volume: 1.0

logging:
  level: "INFO"
  save_audio_clips: false      # 是否保存噪音片段
  clips_dir: "logs/clips"
```

## 使用流程

1. 连接麦克风和蓝牙音箱
2. 运行 `python main.py`
3. 系统自动学习背景噪声（约10秒）
4. 开始实时监测，检测到噪音时自动播放提醒
5. 查看日志文件了解检测历史

## 注意事项

- 首次使用需校准：在安静环境下运行，让系统学习背景噪声
- 阈值参数可能需要根据实际环境微调
- 蓝牙音箱可能有延迟，建议使用有线音箱或低延迟蓝牙音箱
- 请合理使用，避免激化邻里矛盾
